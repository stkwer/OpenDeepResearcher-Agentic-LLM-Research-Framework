# Research Assistant – Agentic LLM

## Overview
This project implements a Research Assistant using an Agentic Large Language
Model approach. The system is designed to assist users in research tasks such
as planning, information gathering, summarization, and evaluation.

## Architecture
The framework follows a Planner–Executor–Critic architecture:

- Planner Agent: Breaks research queries into smaller tasks
- Executor Agent: Performs research and analysis
- Critic Agent: Evaluates and refines the results
- Orchestrator: Manages agent interactions

## Features
- Research task planning
- Automated summarization
- Self-evaluation of outputs
- Modular and extensible design

## Tech Stack
- Python
- Large Language Models (LLMs)
- Agent-based system design

## Use Cases
- Academic research assistance
- Literature review support
- AI research experimentation
