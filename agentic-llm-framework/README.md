# Agentic LLM Framework

A modular Agentic Large Language Model framework implementing
Planner–Executor–Critic architecture.

## Features
- Planner Agent for task decomposition
- Executor Agent for action execution
- Critic Agent for self-evaluation
- Central Orchestrator
- Scalable & extensible design

## Tech Stack
- Python
- LLM APIs
- Modular Agent Design

## Use Case
Autonomous AI workflows and research-based agent systems.
