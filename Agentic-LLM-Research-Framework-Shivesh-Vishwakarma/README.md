# Agentic LLM Research Framework  
**Author:** Shivesh Vishwakarma  

## Overview
The Agentic LLM Research Framework is a modular and extensible system
designed to explore autonomous agent behavior using Large Language Models.
It follows a Planner–Executor–Critic architecture to enable reasoning,
execution, and self-evaluation in AI workflows.

## Architecture
- **Planner Agent** – Breaks down complex tasks into structured steps
- **Executor Agent** – Performs actions based on the generated plan
- **Critic Agent** – Evaluates outputs and provides feedback
- **Orchestrator** – Coordinates communication between agents

## Key Features
- Modular agent-based design
- Clear separation of planning, execution, and evaluation
- Scalable for research and experimentation
- Suitable for autonomous AI workflows

## Tech Stack
- Python
- Large Language Model APIs
- Agent-Oriented Design

## Use Cases
- Research on autonomous AI agents
- Task planning and execution systems
- LLM-based reasoning pipelines
