# AI Research Assistant

An agentic LLM-based research paper generation system using:
- Planner Agent
- Search Agent
- Writer Agent
- Local LLM (Qwen / Ollama)

## Features
- Research paper–style output
- In-text citations
- Final references section
- Markdown and PDF export
- Streamlit UI

## Tech Stack
- Python
- LangGraph
- Streamlit
- Local LLM (Qwen / Ollama)
